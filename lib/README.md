# HyperSelf (MVP)
Gerçek hayatını oyunlaştır: aktivite gir → XP → level.

## Hızlı Başlangıç
1) Vercel env: SUPABASE URL & ANON KEY
2) Supabase SQL'i çalıştır (profiles, activities)
3) / (landing), /(auth)/signup, /(auth)/login, /(app)/dashboard, /(app)/log

## Kategoriler
Sleep, Sport, Social, Learn, Build, Rest

## XP
minutes × multiplier × intensity_bonus
